Loop:
	for(int i = 0;i < n;i++){
		执行语句;
	}
	while(i < n){
		执行语句;
		i++;
	}
	do{
		执行语句;
		i++;
	}while (i<n);
for循环、while循环先判断再执行，do-while先执行后判断；
知道循环几次使用for;不知道则使用while；
for循环中，控制循环的变量只在循环中使用.while循环中，控制循环的变量在循环后还可以继续使用.

死循环：可用作服务器程序等
	for( ; ; ){
		执行语句;
	}
	while (true){			[默认]
		执行语句;
	}
	do{
		执行语句;
	}while (true);

break:
	跳出并结束当前所在循环的执行.
	只能用于结束所在循环，或者结束所在sw1tch分支的执行.
continue:
	用于跳出当前循环的当次执行，直接进入循环的下一次执行.
	只能在循环中进行使用.
