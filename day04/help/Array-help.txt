数组:
	静态初始化数组的格式：
		数据类型[] 数组名 = new 数据类型[]{元素1, 元素2, 元素3,····}
		数据类型[] 数组名 = {元素1， 元素2， 元素3，···}	简化
		数据类型 数组名[] = {元素1， 元素2， 元素3，···}
	数组变量存放数组对象存放的地址
	动态初始化数组的格式:
		数据类型[] 数组名 = new 数据类型[长度]
		默认值：	
			基本类型：
				byte、short、char、int、long		0
				float、double				0.0
				boolean					false
			引用数据类型：
				类、接口、数组、String			null
多个变量指向同一个数组
	数组变量存放同一个数组对象的地址
如果某个数组变量存储的地址是null,那么该变量将不再指向任何数组对象
