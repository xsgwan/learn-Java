类型转换:
	自动类型转换:
		类型范围小 -> 类型范围大(变量)	[直接赋值]
	表达式的自动类型转换：(不同类型变量/数据一起运算)
		在表达式中，小范围类型的变量 -> 较大范围的类型，再参与运算。
		注意
		{
			表达式的最终结果类型由表达式中的最高类型决定。
			在表达式中，byte、short、char是直接转换成int类型参与运算的。
		}
	强制类型转换:
		类型范围大 -> 类型范围小(变量)	[报错]
		数据类型 变量2 = (数据类型) 变量1、数据
		注意
		{
			强制类型转换可能造成数据（丢失）溢出；
			浮点型强转成整型，直接丢掉小数部分，保留整数部分返回
		}
