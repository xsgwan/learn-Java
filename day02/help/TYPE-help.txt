数据类型的分类：
	基本数据类型：(4大类8种)
		整型：
		{
			byte	1字节	-128~127
			short	2字节	-32768~32767
			int	4字节	-2147483648~2147483647				[默认]
			long	8字节	-9223372036854775808~9223372036854775807
		}
		浮点型：
		{
			float	4字节	1.401298*10^(-45)~3.4028235*10^(+38)
			double	8字节	4.9000000*10(-324)~1.797693*10(+308)		[默认]
		}
		字符型：
			char	2字节	0~65535
		布尔型：
			boolean	1字节	true，false
	引用数据类型：
		String
