API:(Application Programming Interface) 应用程序编程接口
	package:
		包是用来分门别类的管理各种不同程序的，类似于文件夹，建包有利于程序的管理和维护。
		package com.gyy.pkg;
		注意事项:
			如果当前程序中，要调用自己所在包下的其他程序，可以直接调用。
				（同一个包下的类，互相可以直接调用)
			如果当前程序中，要调用其他包下的程序，则必须在当前程序中导包，才可以访问！
				导包格式：import包名.类名；
			如果当前程序中，要调用java提供的程序，也需要先导包才可以使用；
				但是java.lang包下的程序是不需要我们导包的，可以直接使用。
			如果当前程序中，要调用多个不同包下的程序，而这些程序名正好一样，
				此时默认只能导入一个程序，另一个程序必须带包名访问。
